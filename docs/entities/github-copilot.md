# GitHub Copilot

## 简介

GitHub 推出的 AI 代码助手，由 OpenAI GPT-4 提供支持。直接在编辑器中提供代码补全和生成。

## 功能

- **代码补全**：根据上下文自动补全代码
- **多语言支持**：Python, JavaScript, TypeScript, Go, Rust 等
- **注释生成代码**：描述需求，自动生成代码
- **代码解释**：选中代码，解释功能
- **测试生成**：自动生成单元测试
- **GitHub Copilot Chat**：对话式辅助（需订阅）

## 定价

- **免费版**：每月 50 次代码补全
- **Copilot Pro**：$10/月，无限补全 + Chat
- **Copilot Business**：$19/月/用户，含企业功能
- **Copilot Enterprise**：$39/月/用户，个性化功能

## 与 VS Code 集成

主要通过官方扩展：
- GitHub Copilot
- GitHub Copilot Chat
