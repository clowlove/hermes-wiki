# Obsidian Software

## 简介

Obsidian 是一款本地优先的笔记和知识管理应用，基于 Markdown 文件。

## 核心特性

- **本地存储** - 文件存储在本地，不过渡到云端
- **双向链接** - `[[wiki-links]]` 语法
- **图形视图** - 可视化知识网络
- **插件生态** - 丰富的社区插件
