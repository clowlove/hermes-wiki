# Zettelkasten

## 什么是 Zettelkasten

Zettelkasten（德语"卡片盒"）是一种个人知识管理方法论，由德国社会学家 Niklas Luhmann 创立。他用这种方法写出了 70 多本书、400 多篇论文。

## 核心原则

1. **原子化** - 每条笔记只包含一个想法
2. **独立卡片** - 每张卡片可独立理解
3. **双向链接** - 卡片之间相互引用
4. **自己语言** - 用自己的话重述
5. **持续积累** - 长期坚持，逐步形成网络

## 与传统笔记的区别

| 维度 | 传统笔记 | Zettelkasten |
|------|----------|--------------|
| 组织方式 | 层级分类 | 网络连接 |
| 复用性 | 低 | 高 |
| 知识转化 | 复制粘贴 | 消化重述 |
| 长期价值 | 难以复用 | 持续增值 |

## 实现工具

- **Obsidian** - 最流行的 Zettelkasten 工具
- **Logseq** - 大纲式 Zettelkasten
- **Roam Research** - 早期推广者
- **Zettlr** - 纯 Markdown Zettelkasten

## 在 LLM Wiki 中的应用

结合 [[LLM Wiki]] 和 [[Obsidian]]：
- Hermes 负责读取、拆解、建页
- Obsidian 负责查看、审阅、图谱展示
- 同一 Wiki 文件夹，双向同步

## Zettelkasten 流程

```
阅读/学习 → 理解 → 用自己话写卡片 → 链接到相关卡片 → 形成知识网络
```
