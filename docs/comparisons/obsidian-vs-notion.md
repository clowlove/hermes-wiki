# Obsidian vs Notion

## 核心对比

| 维度 | Obsidian | Notion |
|------|----------|--------|
| 数据存储 | 本地 Markdown | 云端数据库 |
| 隐私 | 数据完全本地 | 数据在服务器 |
| 双向链接 | ✅ 完整 | ❌ 有限 |
| 图谱视图 | ✅ 成熟 | ❌ 无 |
| 插件生态 | 1000+ | ❌ 无 |
| 团队协作 | 弱 | ✅ 强 |
| 中文支持 | ✅ | ✅ |
| 免费版 | 核心功能免费 | 功能受限 |

## 适用场景

### 推荐 Obsidian
- 个人知识管理
- 第二大脑
- [[LLM Wiki]]
- 需要数据隐私
- 喜欢本地优先

### 推荐 Notion
- 团队协作
- 项目管理
- 数据库需求
- 协作编辑
- 不介意云端

## 双链能力

**Obsidian**：
- `[[wikilink]]` 双向链接
- Backlinks 面板
- Graph View 图谱
- 完整的知识网络

**Notion**：
- `@页面` 提及
- 无真正双链
- 无图谱视图

## 数据迁移

Obsidian：纯 Markdown 文件，任何工具都可打开
Notion：专有格式，导出为 Markdown 或 HTML

## 结论

对于**个人知识管理**和 **LLM Wiki** 场景，**Obsidian 更适合**。

对于**团队协作**和**项目管理**，Notion 更有优势。
