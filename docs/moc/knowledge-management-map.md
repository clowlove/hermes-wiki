# 知识管理地图

## 概述

知识管理 (Knowledge Management) 是组织、存储、检索和分享知识的系统。本地图帮助理解建立个人知识库的方法论。

## 核心概念

### 知识组织
- [[Zettelkasten]] - 卡片盒笔记法
- [[MOC]] - 主题地图
- [[双链交叉引用]] - 知识互联
- [[Source-first]] - 溯源原则

### 笔记方法
- [[Zettelkasten]] - 原子化笔记
- Evergreen Notes - 常青笔记
- [[Obsidian]] - 笔记工具
- [[Logseq]] - 大纲笔记工具

## 建立知识库步骤

1. **选择工具**：推荐 Obsidian 或 Logseq
2. **设置 Wiki 结构**：参考 [[SCHEMA]]
3. **开始捕获**：随手记笔记，不求完美
4. **建立链接**：重要概念使用 `[[wikilink]]`
5. **创建 MOC**：组织主题知识
6. **定期回顾**：迭代优化知识网络
