# AI Agent 最佳实践

## 设计原则

### 1. 明确职责范围
- Agent 应该做一件事做好
- 避免功能过度集中
- 设计清晰的输入输出

### 2. 上下文管理
- 有效利用 [[Memory System]]
- 控制上下文长度
- 重要信息放在开头/结尾

### 3. 错误处理
- 添加重试机制
- 优雅降级策略
- 错误日志和恢复

## 提示词工程

### System Prompt 结构
```
1. 角色定义
2. 核心能力
3. 行为约束
4. 输出格式
5. 示例（可选）
```

### 技巧
- 使用 [[Chain of Thought]]
- Few-shot 示例
- 明确步骤指引
- 限制返回长度

## 工具使用

### 工具设计
- 单一职责
- 清晰的参数定义
- 合理的返回值
- 错误信息友好

### 常用工具
- 网络搜索
- 文件读写
- 代码执行
- API 调用
