# Logseq

## 简介

Logseq 是一款开源的大纲笔记工具，专注于双向链接和知识管理。与 Obsidian 类似，但更强调大纲和结构。

## 核心特性

| 特性 | 说明 |
|------|------|
| 大纲式编辑 | 基于块 (block) 的组织方式 |
| 双向链接 | 完整的 `[[wikilink]]` 支持 |
| 本地优先 | 数据存储在本地 |
| 图谱视图 | 知识网络可视化 |
| 开源 | MIT 协议 |
| 中文优化 | 对中文用户友好 |

## 与 Obsidian 对比

| 维度 | Obsidian | Logseq |
|------|----------|--------|
| 编辑方式 | 页面为主 | 大纲/块为主 |
| 开源 | ❌ | ✅ |
| 插件生态 | 丰富 | 一般 |
| 移动端 | ✅ | ✅ |
| 同步 | 付费服务 | 开源 Sync |
| 学习曲线 | 中等 | 低 |

## 使用场景

- 个人知识管理
- 间隙笔记 (Zettelkasten)
- 项目管理
- 日程规划
