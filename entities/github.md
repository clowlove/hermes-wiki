# GitHub

## 简介

全球最大的代码托管平台，1900 万开发者，提供 Git 仓库托管、代码审查、项目管理、CI/CD、GitHub Actions 等功能。

## 核心功能

| 功能 | 说明 |
|------|------|
| Repository | 代码仓库，支持 public/private |
| Issues | 问题跟踪和项目管理 |
| Pull Requests | 代码审查和合并 |
| Actions | CI/CD 自动化工作流 |
| Packages | 包托管 |
| Codespaces | 云端开发环境 |
| Copilot | AI 代码助手 |

## 与 Hermes 的集成

Hermes 可以通过 `gh` CLI 与 GitHub 交互：

- 管理仓库和文件
- 创建和管理 Issues/PRs
- 触发和监控 Actions
- 代码审查
