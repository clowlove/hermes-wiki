# Obsidian

## 简介

Obsidian 是一款本地优先的知识管理和笔记应用，基于 Markdown 文件，支持双向链接和图谱可视化。

## 核心特性

| 特性 | 说明 |
|------|------|
| 本地存储 | Markdown 文件，隐私安全 |
| 双向链接 | `[[wikilink]]` 实现笔记互联 |
| 图谱视图 | 可视化知识网络 |
| 插件生态 | 1000+ 社区插件 |
| 多种主题 | 高度可定制 |
| 免费版功能强大 | 足够个人使用 |

## 价格

- **个人免费版**：核心功能全部免费
- **Obsidian Publish**：$8/月，发布在线笔记
- **Obsidian Sync**：$4/月，云同步
